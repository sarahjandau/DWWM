Règles du jeu

Dragons Dynasty est un jeu de stratégie au tour par tour où les joueurs contrôlent des dragons pour conquérir des territoires et vaincre les autres joueurs.

Sélection du dragon :

Chaque joueur commence avec un dragon de base.


Tours par tours :

Le jeu se déroule en tours. Pendant chaque tour, les joueurs peuvent effectuer une action (attaquer ou utiliser une compétence spéciale).


Combat

Le vainqueur est déterminé en fonction de son nombre de vie.


Victoire

Le joueur qui parvient à éliminer les dragons des autres joueurs remporte la partie.
